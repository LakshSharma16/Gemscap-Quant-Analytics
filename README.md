# Quantitative Analytics Dashboard

## Objective
This project is built as part of the Quant Developer Evaluation Assignment.
It demonstrates an end-to-end analytical system from market data ingestion
to quantitative analytics and interactive visualization.

## Features
- Real-time market data ingestion
- Timeframe-based resampling (1s, 1m, 5m)
- Hedge ratio (OLS regression)
- Spread and Z-score analysis
- ADF stationarity test
- Rolling correlation
- Interactive Plotly charts

## Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Statsmodels
- Plotly

## How to Run
```bash

streamlit run app.py
